# نظام إدارة المكتبة - Library Management System 📚

هذا المشروع هو تطبيق مكتبي بلغة ++C يعتمد على مبادئ البرمجة كائنية التوجه (OOP).

### 🛠 التقنيات المستخدمة:
* OOP: استخدام الوراثة (Inheritance) والكلاسات لتنظيم البيانات.
* File Handling: حفظ واسترجاع بيانات الكتب من ملفات نصية (.txt).
* Logic: معالجة النصوص وتحويلها لسجلات (String Splitting).

### 🚀 المميزات:
1. إضافة كتب جديدة مع حفظ (العنوان، المؤلف، عدد الصفحات).
2. عرض جميع الكتب المخزنة في جدول منسق واحترافي.
